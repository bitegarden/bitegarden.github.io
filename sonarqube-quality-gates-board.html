What is Quality Gates Boards for SonarQube™?

Quality Gates Boards for SonarQube™ is a plugin that adds a page to the SonarQube™ project so you can validate if the code meets a set of quality thresholds.

This product has been developed because SonarQube™ allows you to validate your code against only a quality threshold. With this plugin you can validate your code against more than one threshold.

By default, this page will show all the quality thresholds and all the conditions so you can see how many Quality Gates it meets.

 

Set up multiple quality thresholds in your project

This product allows you to define the set of thresholds in the administration of the plugin, with the aim that you have all the information on a single page.

With Quality Gates Board for SonarQube™ you can easily identify which conditions are met and which are not.

Customize thresholds and their presentation

In this plugin you can customize the title of the quality thresholds so that you can include information relevant to your organization. A useful feature, for example, to show different quality levels, such as "Gold", "Silver" and "Bronze", according to the quality metrics defined in your team.
