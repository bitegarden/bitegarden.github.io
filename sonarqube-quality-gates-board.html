Quality Gates Board for SonarQube™

Provides a board in your project space to check if you pass or fail a given set of quality gates

TRY NOW    BUY NOW

Validate your code against multiple quality gates

SonarQube™ allows you to check your code against one quality gate, and only one.

Quality Gates Board provides a new page in your project space to check if you pass or fail a given set of quality gates.

By default the board will show all the quality gates along with all the conditions passing and failing.

 

 

Setup the quality gates to check

Define a set of quality gates in your plugin settings and in one page you will have the information for those quality gates. You will know which one is passing or failing and the conditions.

This will help you to promote your application to the next compliance level.

Customize header title

You can edit your header title to display any related information in your organization about the quality gates board.

This will help your users to understand different quality gate levels (for instance: gold, silver, bronce, ...).
