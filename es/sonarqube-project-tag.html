Etiqueta tus proyectos de manera automática

¿Alguna vez has tratado de automatizar el on-boarding de proyectos en SonarQube™ usando etiquetas?

SonarQube™ tiene una potente gestión de etiquetas/tags en el espacio de proyectos, pero la única manera de crear nuevas etiquetas es a través del propio sonar y de manera manual. Crear etiquetas manualmente es muy difícil cuando tienes muchos proyectos. Con nuestro plugin, podrás configurar etiquetas de proyecto de forma fácil y automática. Simplemente proporciona la etiqueta a través de una propiedad de línea de comando en tu análisis y el plugin hará el resto.

Cuando ya tienes metadatos del proyecto, querrás usarlo durante el análisis del proyecto (en tu herramienta de CI como Jenkins). Todo está automatizado y la etiqueta del proyecto se configura utilizando los metadatos y está disponible lo antes posible.

Uso y configuración

Para automatizar la creación de etiquetas en SonarQube™:

Escribe la etiqueta a través de una propiedad de línea de comando: solo tienes que añadir esta propiedad 

-Dbitegarden.project.tag=yourtag
a cualquiera de tus scanners y el plugin captará la información y la guardará en SonarQube™.

Es importante saber que las etiquetas en SonarQube™ solo aceptan los siguientes caracteres: 

a-z, 0-9, '+', '-', '#', '.'

En la configuración del administrador, lanza el proceso para etiquetar todos los proyectos: una vez que tus proyectos han sido analizados usando la propiedad, puedes lanzar el proceso que etiquetará los proyectos por ti. Ve a Administration -> Configuration -> Project Tag y lánzalo

Las etiquetas de sistema y las personalizadas se conservan: el proceso no eliminará ninguna etiqueta y solo agregará la etiqueta utilizada durante el análisis.
