Conoce la calificación de más características de calidad software

SonarQube™ tiene un potente modelo de calidad basado en tres grandes características: fiabilidad, mantenibilidad y seguridad.Pero a veces, es necesario ir un poco más allá. ¿Qué pasa con el rendimiento? ¿Y la gestión de errores? ¿Qué calidad tiene mi proyecto con respecto a la usabilidad?

Con este plugin podrás obtener la calificación para todas esas características en un informe adicional a partir de las etiquetas de las reglas de SonarQube™

Rating Report Plugin for SonarQube™ calcula la calificación (A, B, C...) en SonarQube™ para cada uno de los tags (etiquetas) y genera un informe adicional para tu proyecto. Permite así evaluar la calidad del código más allá de la seguridad, la fiabilidad y la mantenibilidad.

Características

Con este plugin de SonarQube™ podrás conocer nueva información para cada etiqueta:

Tag Rating: la calificación de cada etiqueta está basada en la severidad, de la misma manera que en las calificaciones de seguridad y fiabilidad

Total number of issues per Tag: podrás ver en el informe el número de evidencias que tiene cada etiqueta

Issues per severity per Tag: proporciona una forma de conocer el riesgo de todas las etiquetas por gravedad
