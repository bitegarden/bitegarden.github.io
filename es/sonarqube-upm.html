Universal Plugin Manager for SonarQube™

Proporciona una interfaz sencilla para gestionar tus plugins

Para Administradores por Desarrolladores

¿Cuántas veces quieres instalar un plugin personalizado en tu instancia de SonarQube™ y tienes que esperar hasta que el equipo de operaciones/sistemas copie el "fichero" a tu servidor dentro de la carpeta "extensions/plugins"?

¡Esta será la última vez que lo hagas! Universal Plugin Manager (UPM) es una herramienta para administrar plugins en SonarQube™.

Puedes usar el UPM para instalar plugins individuales (ficheros "jar") a través de la interfaz de administración de SonarQube™.

Con tan solo unos clicks podrás navegar los plugins de bitegarden y tus plugins personalizados sin abandonar la interfaz de administración de SonarQube™.

Puedes instalar plugins a través de la carga de ficheros. La carga de ficheros instala el plugin como un fichero JAR. El fichero debe contener todos los recursos y código necesarios para ejecutarse. La carga de ficheros te permite instalar plugins personalizados o cualquier otro plugin que no esté disponible en el Marketplace de SonarQube™. Puedes también utilizarlo para instalar plugins oficiales detrás de un firewall y desconectado de internet.

¡No tendrás que acceder al servidor de SonarQube™ nunca más para instalar un plugin personalizado!

Descarga Gratis

Plugins destacados de bitegarden

Universal Plugin Manager for SonarQube™ incluye una nueva sección para gestionar todos los plugins desarrollados por bitegarden.

De esta forma con solo instalar el UPM ya podrás usarlo para instalar todos nuestro plugins. También puedes actualizar a nuevas versiones o eliminar aquellos que no utilices desde la interfaz de administración.

Además también incluye la funcionalidad para auto-actualizarse, avisando de la disponibilidad de nueva versión. Podrás directamente actualizar desde la página de administración del UPM.

Plugins gestionados por el usuario

Puedes instalar cualquier plugin cargandolo directamente en tu plataforma SonarQube™. Tienes que usar este método para instalar plugins que no estén disponibles en la sección destacada o en el Marketplace oficial, como un plugin desarrollado por ti o por si quieres instalar una versión antigua de otros plugins.

Advertencia de seguridad: los plugins son muy potentes, y pueden cambiar el comportamiento de casi cualquier funcionalidad de tu plataforma SonarQube™. Es muy importante que confies en los plugins antes de instalarlos. Siempre debes tener cuidado de donde viene (y de quién es) el plugin a instalar.
