SonarQube™ / SonarCloud™
plugins y productos

Construimos plugins de SonarQube™ y SonarCloud™ para mejorar tu ecosistema de inspección continua

Report Plugin for SonarCloud™

Genera informes en PDF o Excel de tus proyectos analizados en SonarCloud™

Report Plugin for SonarQube™

Genera un informe a nivel proyecto y personalizado. Reporte generado en ODT, Excel o PDF y disponible para descargar

Overview Report Plugin for SonarCloud™

Genera un PDF con la visión global de la calidad y seguridad de tu organización en SonarCloud™

Overview Report Plugin for SonarQube™

Permite agregar proyectos en SonarQube™ para tener una vista general de las métricas de calidad de código en una única vista

Security Report for SonarCloud™

Genera un informe en PDF para validar estándares como OWASP Top 10 o CWE SANS Top 25 de tu proyecto en SonarCloud™

Security Plugin for SonarQube™

Proporciona información sobre estándares de seguridad como OWASP incluyendo factores de riesgo y vulnerabilidades

Control Center for SonarQube™

Centro de Control para administradores de SonarQube™

SQALE Plugin for SonarQube™

El método SQALE aplicado en SonarQube™ permite gestionar la deuda técnica por características de calidad

Rating Report Plugin for SonarQube™

Añade un informe adicional usando la información de las etiquetas en SonarQube™

Project Tag for SonarQube™

Establece tags/etiquetas automáticas en el onboarding de proyectos en SonarQube™

Metrics Pack Plugin for SonarQube™

Pack de métricas adicionales para SonarQube™. Incluye métricas relacionadas con RCI, dead code, etc.

Universal Plugin Manager for SonarQube™

La forma más sencilla de instalar plugins desde la interfaz de administración
