Genera informes de métricas con SonarQube

¿Necesitas exportar informes de SonarQube de manera rápida y sencilla? Con el Report Plugin for SonarQube™ podrás generar reportes completos de tus proyectos en PDF, ODT y EXCEL, listos para presentaciones, auditorías o análisis de calidad de código.

Con el Report Plugin for SonarQube™ puedes descargar informes en PDF claros y profesionales con todas las métricas clave sobre la calidad del código de tu proyecto. Todo de forma rápida y sencilla, directamente desde la interfaz de SonarQube, sin configuraciones adicionales ni pasos extra.

Además, el plugin te permite generar informes personalizados a partir de plantillas de Open Document (ODT) y EXCEL. Estos informes recopilan automáticamente las métricas más relevantes de tu proyecto en SonarQube y las presentan en un documento editable, para que puedas añadir información adicional para auditorías o revisiones internas.

Características

Las características principales del plugin incluyen:

- Informe ejecutivo en PDF: todas las métricas importantes en una sola página.
- Informe de evidencias resumido: incluye bugs, vulnerabilidades y code smells.
- Informe completo de evidencias: todos los detalles de tu proyecto en un PDF listo para análisis.
- Plantillas ODT personalizables: modifica cabeceras, tablas, textos, títulos, portadas, imágenes y más.
- Opciones de personalización: agrega logos en pies de página y adapta el diseño a tus necesidades.

Y además:

- Permite generar los informes de cualquier rama de tu proyecto
- Compatible con todos los lenguajes y tecnologías de SonarQube Server, incluyendo todos los plugins de terceros.
- Los informes se generan al instante y se pueden descargar desde la interfaz de usuario de SonarQube o a través de un endpoint de un servicio web para integrarlo con otras herramientas.

FAQ - Preguntas frecuentes

¿Puedo generar informes de cualquier rama de mi proyecto?
Sí, el plugin permite seleccionar cualquier rama de tu proyecto en SonarQube y generar un informe actualizado en el momento.

¿Qué diferencias hay entre los informes PDF y ODT?
El PDF es ideal para compartir resultados finales con tu equipo o clientes, mientras que el ODT te permite editar el documento y añadir comentarios, conclusiones o datos de auditoría.

¿Los informes incluyen la información de cobertura y duplicaciones de código?
Sí, los informes pueden incluir métricas de cobertura, duplicaciones, bugs, vulnerabilidades y code smells, entre otras.

¿Funciona con todas las ediciones de SonarQube™ (Community, Developer, Enterprise y Data Center)?
Sí, el plugin es compatible con todas las ediciones, aunque algunas métricas avanzadas pueden depender de la edición de SonarQube™ que utilices.

¿Se puede incluir el logo o nombre de mi empresa en los informes?
Sí, puedes personalizar los informes con tu logo en el pie de página o portada, y adaptar los estilos en las plantillas ODT.
