¿Qué es Quality Gates Boards for SonarQube™?

Quality Gates Boards for SonarQube™ es un plugin que añade una página en el proyecto de SonarQube™ para que puedas validar si el código cumple contra un conjunto de umbrales de calidad.

Este producto se ha desarrollado porque SonarQube™ permite validar tu código contra solo un umbral de calidad. Con este plugin podrás validar tu código contra más de un umbral.

Por defecto, esta página mostrará todos los umbrales de calidad y todas las condiciones para que puedas cuantos Quality Gates cumple.

 

Configura múltiples umbrales de calidad en tu proyecto

Es producto te permite definir el conjunto de umbrales en la administración del plugin, con el objetivo de que tengas toda la información en una sola página.

Con Quality Gates Board for SonarQube™ podrás identificar fácilmente qué condiciones se cumplen y cuáles no.

Personaliza los umbrales y su presentación

En este plugin puedes personalizar el título de los umbrales de calidad para que puedas incluir información relevante para tu organización. Una caractertística útil,por ejemplo, para mostrar diferentes niveles de calidad, como "Oro", "Plata", y "Bronce", según las métricas de calidad definidas en tu equipo.
