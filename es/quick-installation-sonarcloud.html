Guía de instalación rápida

Esta guía te enseña a utilizar los productos para SonarCloud

Acabas de descargar un archivo "jar" ejecutable automático que contiene el producto (aplicación independiente) para SonarCloud™. Requiere Java 8 o posterior para ejecutarse.

Ejecute el comando con la opción --help para ver cómo usarlo (cambie bitegarden-product.jar al archivo que acaba de descargar): 

java -jar bitegarden-product.jar --help


NOTA: El producto incluye una licencia de prueba de 2 semanas para que lo evalues. No necesitas hacer nada más.
