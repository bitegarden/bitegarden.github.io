SonarQube™ / SonarCloud™
Plugins and Products

Sviluppiamo plugins per SonarQube™ e SonarCloud™ per migliorare il tuo ecosistema di integrazione continua

Report Plugin for SonarCloud™

Permette di scaricare il report di SonarCloud™. Il report può essere generato in PDF o Excel.

Report Plugin for SonarQube™

Permette di scaricare il report di SonarQube™ con pochi semplici click. Il report può essere generato in PDF o Excel o in un ODT personalizzabile.

Overview Report Plugin for SonarCloud™

Genera un PDF con la visione globale della qualità e della sicurezza della tua organizzazione in SonarCloud™

Overview Report Plugin for SonarQube™

Permette di aggiungere diversi progetti di SonarQube™ ed avere una vista con tutte le metriche di qualità del codice.

Security Report for SonarCloud™

Fornisce informazioni sugli standard come OWASP Top 10 o CWE SANS Top 25, inclusi fattori di rischio e vulnerabilità.

Security Plugin for SonarQube™

Fornisce informazioni sugli standard di sicurezza come OWASP, inclusi fattori di rischio e vulnerabilità.

Dependency Check for SonarCloud

Sincronizza le informazioni Dependency Check con il tuo SonarCloud

Dependency Track for SonarQube™

Sincronizza le informazioni Dependency Track con la tua istanza SonarQube

Control Center for SonarQube™

Centro di controllo per gli amministratori SonarQube™

SQALE Plugin for SonarQube™

Il metodo SQALE applicato a SonarQubee™ permette di gestire e ridurre il debito tecnico.

Rating Report Plugin for SonarQube™

Crea un nuovo report utilizzando le informazioni delle etichette presenti in SonarQube™.

Project Tag for SonarQube™

Imposta tags/etichette automatici nell'onboarding dei progetti in SonarQube™

Metrics Pack Plugin for SonarQube™

Pack di metriche aggiuntive per SonarQube™. Include metriche relative a RCI, dead code, ecc.

Universal Plugin Manager for SonarQube™

Permette di installare o disinstallare plugins per SonarQube direttamente dall'interfaccia di amministrazione.

Generic Issue Data Exporter for SonarQube™

Importa problemi SonarQube nella tua istanza SonarCloud
