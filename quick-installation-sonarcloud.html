Quick Installation Guide

This guides helps you to start using our SonarCloud™ products

You have downloaded an auto executable "jar" file containing the product (standalone app) for SonarCloud™. It requires Java 8 or later to run.

Run the command with the --help option to see how to use it (change bitegarden-product.jar to the file you have just download): 

java -jar bitegarden-product.jar --help


NOTE: The product includes the trial license for 2 weeks for evaluation. You don't have to setup any license key.
