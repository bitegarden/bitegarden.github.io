The main new feature of the new version of **Control Center for SonarQube&trade; Server** is the beta release of the **Audit module**, a new functionality designed to perform automatic checks on your SonarQube instance and help identify problematic or improvable configurations.

In this first Audit version, we have **included five initial checks** focused on security, maintenance, and license optimization:

Default admin credentials check. <br>

Online LTA version check. <br>

Latest LTA version check. <br>

Official SonarQube distribution check. <br>

Commercial SonarQube edition without branch usage check. <br>

<img width="100%" src="/img/sonarqube-control-center/sonarqube-control-center-audit.png" alt="Control Center SonarQube Checks Audit">
<br>

We would like to remind you that this plugin allows you to manage projects, users, and hotspots in a massive and centralized way, greatly simplifying daily administrative tasks within SonarQube.

If you are already using [Universal Plugin Manager for SonarQube&trade; Server](/es/sonarqube-upm), updating to the new version of Control Center for SonarQube™ Server is as simple as accessing your instance administration panel and clicking the update button. And if you have not tried it yet, you can download and install it directly from the [Marketplace de bitegarden](https://marketplace.bitegarden.com/).
