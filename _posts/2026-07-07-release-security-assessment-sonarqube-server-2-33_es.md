¡Nueva release del Security Assessment for SonarQube&trade; Server! En esta **versión 2.33** añadimos un **nuevo estándar de seguridad** y **mejoras de rendimiento** que reducen el tiempo de generación de determinados informes.

El nuevo estándar es **STIG ASD** (Application Security and Development Security Technical Implementation Guide), desarrollado por la DISA (Departamento de Defensa de EE. UU.). Este marco define requisitos de seguridad para el desarrollo de software con el objetivo de proteger los datos y reducir el riesgo de ciberataques.

<img width="100%" src="/img/sonarqube-security/security-assessment-sonarqube-server-stig-asd.png" alt="Report STIG in SonarQube">
<br><br>

Esta incorporación también incluye la descarga del informe en PDF:
<br>

<img width="80%" src="/img/sonarqube-security/security-assessment-sonarqube-server-stig-asd-pdf.png" alt="STIG PDF Report">
<br><br>

Desde esta versión ya puedes evaluar tu código frente a STIG ASD, que se incorpora a los **estándares ya disponibles**: OWASP Top 10, OWASP ASVS, CWE Top 25, CWE On the Cusp, CWE/KEV e ISO 5055.
