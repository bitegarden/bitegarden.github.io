Acabamos de lanzar una nueva versión de **bitegarden Security Assessment for SonarQube&trade; Server**, con la que ya puedes medir tu código contra lo último de seguridad, sin salir de tu flujo habitual en SonarQube Server: **OWASP Top 10 2025 y OWASP ASVS 5.0.0**.

<h2>¿Qué cambia realmente? </h2>

Con esta versión puedes evaluar **el cumplimiento de tus proyectos frente a OWASP Top 10 2025**, alineándote con las amenazas más actuales. También se **incorpora OWASP ASVS 5.0.0**, con un selector que te permite elegir la versión del estándar, igual que eliges ramas o periodos en otros reportes. Con esta actualización obtienes una **visión mucho más precisa** de dónde estás y qué te falta para cumplir con los requisitos de seguridad modernos.

<h2>¿Qué hace este plugin?</h2>

bitegarden Security Assessment for SonarQube&trade; Server no es solo un visor de vulnerabilidades. Con este plugin puedes:

- **Reunir todas las vulnerabilidades** detectadas en SonarQube. <br>

- Traducirlas a **estándares reconocidos como OWASP, CWE, WASC y CERT**. <br>

- Tener una **visión detallada, estructurada y visual** del estado de seguridad de tu proyecto. <br>


<h2>¿Cómo actualizar a la nueva versión?</h2>

Si ya estás usando el plugin, sigue estos pasos:

1 - Accede al [Universal Plugin Manager for SonarQube&trade; Server (UPM)](https://marketplace.bitegarden.com/product/bitegardenUniversalPluginManager). <br>

2 - Pulsa actualizar. <br>

3 - ¡Listo! ya estás operando con los nuevos estándares. <br>

<h2>¿Aún no lo has probado?</h2>

Puedes instalar desde aquí el [bitegarden Security Assessment for SonarQube&trade; Server](https://marketplace.bitegarden.com/product/bitegardenSecurity) y configurarlo en tu instancia rápidamente.
