We’ve released a new version of **Control Center for SonarQube&trade; Server**, introducing significant improvements to the Audit module to provide an even more comprehensive view of the platform’s status and configuration.

<h2>New Audit checks for more comprehensive analysis</h2>

This new version **extends review and control capabilities** by adding new checks to the Audit section (beta), organized into different categories:

2 new general checks. <br>
7 new global settings checks. <br>
8 Data Center specific checks. <br>

<img width="100%" src="/img/sonarqube-control-center/sonarqube-control-center-audit-checks.png" alt="Control Center SonarQube Checks Audit">
<br>

With these additions, administrators can identify potentially improvable configurations, review the instance status with greater accuracy, and **maintain administration** best practices much more easily.

Non-applicable checks, such as the Data Center ones in this case, can be **disabled from the plugin configuration** if users do not wish to display them. Likewise, it is also possible to configure a list of trusted **third-party plugins** so they do not appear in the third-party plugins check.

We’d like to remind you that Control Center for SonarQube&trade; Server is the ultimate plugin for administrators who want to manage their SonarQube instance in a faster, more efficient, and centralized way.

<h2>Actualización sencilla desde UPM</h2>

If you already have [Universal Plugin Manager for SonarQube Server](/sonarqube-upm), you only need to access your instance administration panel and click the update button to start using the new version.

And if you have not tried Control Center for SonarQube™ Server yet, you can download and install it on your instance from our[Marketplace](https://marketplace.bitegarden.com/).
