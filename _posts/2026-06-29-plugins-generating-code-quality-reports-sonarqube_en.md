When an organization uses SonarQube™ to analyze software quality, it has access to a **wealth of information** about its projects: vulnerabilities, bugs, code smells, test coverage, code duplication, and technical debt. However, one of the most common needs for development teams and technical leaders is to answer a simple question: **What is the current state of my code quality?**

Answering this question requires more than reviewing individual metrics. It is essential to **have a comprehensive overview, generate clear reports, and access meaningful insights** that enable informed decisions about the evolution of software projects. That's why bitegarden has developed a range of plugins to extend SonarQube™'s capabilities and simplify the generation of code quality reports.

This article is the first part of a series in which we will **explore different bitegarden plugins** and how they help enhance the SonarQube™ experience. In this first installment, we focus on plugins that **generate reports and provide a clear overview** of the current state of software quality.

<h2>Report for SonarQube™ Server: Generate Comprehensive Code Quality Reports</h2>

Report for SonarQube™ Server allows you to **generate detailed reports with the most relevant information** from your projects analyzed in SonarQube™. The plugin is designed for teams that need to share the **current state of their code quality** beyond the SonarQube™ interface, creating documents that can be used for technical reviews, project status meetings, or audits.

With this plugin, **you can generate reports including**:

* Overall project metrics. <br>

* Issues detected during the analysis. <br>

* Software quality status. <br>

* Quality Gate information. <br>

* Maintainability, reliability, and security metrics. <br>

This gives teams an up-to-date "snapshot" of their code quality in a format that is easy to review and share.

If you haven't tried this plugin yet, you can download it for your instance from the [bitegarden Marketplace](https://marketplace.bitegarden.com/product/bitegardenReport).

<h2>Rating Report Plugin: An Easy-to-Understand Software Quality Rating</h2>

The Rating Report Plugin for SonarQube™ **adds a rating (A, B, C...)** for each SonarQube™ tag and generates an additional report for your project. This rating makes it possible to **quickly assess** the quality of different areas of your code while complementing the standard information provided by SonarQube™.

It is particularly useful for:

Comparing different projects. <br>

Quickly identifying areas for improvement. <br>

Creating executive reports. <br>

Communicating code quality to non-technical stakeholders. <br>

You can try it in your instance through [this link](https://marketplace.bitegarden.com/product/bitegardenRatingReport).

<h2>Overview for SonarQube™ Server: A Global View of All Your Projects</h2>

Overview for SonarQube™ Server provides an **aggregated view of code quality** across a SonarQube™ instance, making it easier to identify trends and gain an overall view of your software's health.

This high-level overview helps answer questions such as:

Which projects currently have the lowest code quality? <br>

Where is the greatest amount of technical debt? <br>

Which teams should prioritize improvements? <br>

How is code quality evolving over time? <br>

You can learn more about [this plugin](https://marketplace.bitegarden.com/product/bitegardenPortfolio) on the bitegarden Marketplace.
