Si quieres exportar **informes de los perfiles de calidad y sus reglas activas**, el Report Plugin for SonarQube te permite hacerlo fácilmente. Desde la **versión 2.22**, esta funcionalidad está disponible, y en este artículo te explicamos cómo aprovecharla paso a paso.

<h2>Paso 1: Descargar el Report Plugin for SonarQube</h2>

Para comenzar, descarga el **Report Plugin for SonarQube** a través de [este enlace](/es/sonarqube-report).

Además, te recomendamos consultar las [instrucciones](/es/sonarqube-report-documentation) de instalación. Verás que completar este paso es muy sencillo y rápido.


<h2>Paso 2: Generar los informes de perfiles de calidad</h2>

Una vez finalizada la instalación, ya podrás generar tu informe de perfiles de calidad y reglas activas siguiendo estos pasos:

Accede a tu instancia de SonarQube.

Haz clic en la pestaña **More > Rules Reporting**. Para facilitarte la ubicación, aquí tienes una imagen de ejemplo:

<img width="100%" src="/img/sonarqube-report/sonarqube-report-rules-reporting.png" alt="Rules Reporting Report Plugin for SonarQube Server">
<br>
<br>

Al acceder a Rules Reporting, se abrirá una ventana donde tendrás que seleccionar:

El **lenguaje** del que quieras exportar las reglas activas.

El **perfil de calidad** del que quieras generar el informe.

En la imagen anterior, se ha seleccionado el lenguaje Java y el perfil de calidad Sonar way.

<h2>Paso 3: Descargar el informe</h2>

¡Listo! Una vez seleccionados los parámetros, haz clic en el botón **Download**.

Se descargará un **archivo PDF** que podrás leer, guardar y compartir. Aquí tienes un ejemplo de las dos primeras páginas:

<img width="100%" src="/img/sonarqube-report/sonarqube-report-pdf-quality-gates.png" alt="Informe en PDF perfiles de calidad SonarQube">
<br><br>

El **Report Plugin for SonarQube Server** es muy útil no solo para exportar informes de perfiles de calidad y sus reglas activas y para conocer el **estado de calidad de tu código**. Siguiendo los pasos de instalación, selección de **lenguaje y perfil de calidad, y descarga del informe**, podrás tener toda la información clave de manera rápida y accesible.

Esta funcionalidad te permite **mejorar la trazabilidad, la documentación y el control de tus proyectos de desarrollo**, facilitando la comunicación entre equipos y asegurando que se cumplen los estándares de calidad establecidos.
